a태그

Anchor 태그라고 부른다. 서로 다른 웹 페이지 사이를 이동하거나 내부에서 특정한 위치로 이동할 때 사용한다. a 태그는 하이퍼링크 기능을 제거하더라도 웹 표준을 따르기 위해서 href 속성을 반드시 입력해야 한다. 웹 표준을 지키면서 이동하지 않는 a 태그를 만들 때는 href에 #을 넣어준다. 이를 빈 링크라고 부른다. a태그로 id속성을 활용하여 페이지 내부 이동하려면 href="#id" 식으로 사용한다. 만약 id가 중복되는 경우라면, 먼저 나오는 태그로 이동한다. (id 중복은 웹 표준에 어긋난다.)
